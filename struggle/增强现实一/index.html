增强现实技术（一）
  • Jan 1, 0001
  • 3 minutes read

概念

增强现实（Augmented Reality，简称 AR），是一种实时地计算摄影机影像的位置及角度并加上相应图像的技术，是一种将真实世界信息和虚拟世界信息“无缝”集成的新技术，这种技术的目标是在屏幕上把虚拟世界套在现实世界并进行互动。

常见功能辅助插件

  1. Vuforia SDK Vuforia被认为是全球最广泛使用的AR平台之一,相比较而言最为稳定、成熟，不过所有文档都是英文的，有一丢丢费劲。自2017版本开始，Vuforia SDK开始内置于Unity3D引擎之内，无需额外安装，对于大多数开发者来说确实是好消息。但自Vuforia Engine 8.6开始，又不内置了，同时官网称：Unfortunately, the original version of this article included unclear information about Vuforia Engine. 是的，在使用过程中会有一些“神奇”的问题。

  2. EasyAR EasyAR是国内发展最好的一款SDK了，非常简单实用，对于新手来说很容易上手，并且官方api也是中文版自然容易接入。EasyAR现在拥有一个庞大的家族，从版本4开始，以前已知的EasyAR SDK将被命名为EasyAR Sense，EasyAR Sense是一个独立的SDK，而EasyAR Sense Unity插件将EasyAR Sense功能公开到Unity中。

AR开发准备工作

  1. 安装Unity3D，然后配置电脑的java运行环境，也就是下载Java的jdk，配置系统环境变量，当你在控制台敲Java -version会出现jdk版本信息，说明就OK了，接着去下载androidSDK,并配置系统环境变量，当然你也可以下载Android Stdio，它会自动下载AndroidSDK，也许你会问配置这些干嘛啊，当然是为了打包成APP…

  2. 去EasyAR或者Vuforia下载对应的unity插件，当然有收费版本的，我们选择免费版的就好了，建议下载带有官方案例的包，省好多事，然后注册账号获取APP License Key,成功后你会看到一长串字符串，这个是你需要添加到unity工程中的ARCamera组件中需要，否则工程无法使用sdk。进入unity3d中，导入EasyAR或者Vuforia插件，在左下方project的Assets里。区别来了：EasyAR识别图是直接导入unity3d中的，它的path保存在一个json文件里，这就意味着你想更换识别图，或者想识别多张图的话就可以在这个json文件里更改，而Vuforia识别图需要在你获取key的地方，创建识别图的数据库对象，当然可以创建很多张识别图。

  3. 接下来，在unity里，在左上角的Hierarchy下面，删除mianCamera，添加ARCamera，然后导入识别图数据（两者不一样哦），导入3D模型在识别图上方适当位置，怎么适当呢，这个自己琢磨吧。接下来可以打包运行了，在此之前，你还需要配置Java jdk路径,在edit->preferences->external tools这里哦，接下来file->build setting，选择android应用，点击playersettings，在这里有一些乱七八糟的问题需要设置，具体问题具体分析，遇到了再去百度吧，基本都可以解决，不过package需要填对，具体参照API文档。
